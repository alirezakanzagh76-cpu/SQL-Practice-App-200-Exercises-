# SQL Practice App (200 Exercises)

An offline, beginner-friendly SQL practice desktop app built with **Electron + sql.js (SQLite in WebAssembly)**.  
It includes **200 SQL exercises**, a built-in editor, instant results, and a safe execution mode (read-only queries).

## Features
- ✅ 200 curated SQL exercises
- ✅ Runs fully offline (no server required)
- ✅ Built-in SQL editor + results table
- ✅ Read-only execution safety: only `SELECT` / `WITH` allowed
- ✅ Resizable panes (editor / result / schema)
- ✅ Local SQLite database initialized from SQL scripts
- ✅ Progress tracking (if included in your build)

## Tech Stack
- Electron
- sql.js (SQLite compiled to WebAssembly)
- CodeMirror (editor)
- Chart.js (optional charts)
